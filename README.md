# clankers

A terminal coding agent in Rust. Like [pi](https://pi.dev), but yours to hack on.

## Build

```
cargo build --release
```

## Auth

```
export ANTHROPIC_API_KEY=sk-...
```

## Use

```
clankers                        # interactive TUI
clankers -p "fix the tests"     # one-shot
```
